# Task00

  ## Задание
  Составить словарь расстояний между городами (Moscow, London, Paris), используя координаты.
  
  ## Описание решения
  Используется формула расстояния на плоскости: ((x1 - x2)² + (y1 - y2)²) ** 0.5.
  Создаётся словарь словарей, где ключи — города, а значения — расстояния до остальных городов.
  В результате можно получить расстояние от любого города до другого.
  
  ## Запуск
  cd нужный репозиторий
  python main.py 0
